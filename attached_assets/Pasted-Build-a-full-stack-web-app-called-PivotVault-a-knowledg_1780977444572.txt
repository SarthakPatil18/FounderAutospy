Build a full-stack web app called PivotVault — a knowledge repository for startup failures, styled precisely to match the Vercel design system.

── DESIGN SYSTEM ────────────────────────────────────────────

Fonts (load via Google Fonts):
  - Body/Display: Inter (weights 400, 500, 600) — use font-feature-settings: "ss01", "ss02"
  - Monospace: JetBrains Mono (weight 400)

Color tokens (define as CSS variables in :root):
  --color-ink:           #171717;
  --color-body:          #4d4d4d;
  --color-mute:          #888888;
  --color-canvas:        #ffffff;
  --color-canvas-soft:   #fafafa;
  --color-canvas-soft-2: #f5f5f5;
  --color-hairline:      #ebebeb;
  --color-link:          #0070f3;
  --color-on-primary:    #ffffff;

Spacing scale (4px base):
  --sp-xs: 8px; --sp-sm: 12px; --sp-md: 16px;
  --sp-lg: 24px; --sp-xl: 32px; --sp-2xl: 40px;
  --sp-4xl: 64px; --sp-5xl: 96px;

Border radius:
  --r-sm: 6px; --r-md: 8px; --r-lg: 12px; --r-pill: 100px; --r-full: 9999px;

Elevation (stacked shadows, never single heavy drop):
  --shadow-card: 0px 2px 2px #0000000a, 0px 8px 8px -8px #0000000a, inset 0 0 0 1px #00000014;
  --shadow-float: 0px 2px 2px #0000000a, 0px 8px 16px -4px #0000000a, inset 0 0 0 1px #00000014;

Brand gradient (hero use only, full multi-stop mesh):
  background: linear-gradient(135deg, #007cf0, #00dfd8, #7928ca, #ff0080, #ff4d4d, #f9cb28);

Typography rules:
  - Headlines: Inter 600, sentence-case, letter-spacing -2.4px at 48px hero / -1.28px at 32px sections
  - Body: Inter 400, 16–18px
  - Buttons (marketing): Inter 500, 16px, pill shape (border-radius: var(--r-pill))
  - Nav buttons: Inter 500, 14px, border-radius: var(--r-sm) 6px, height 28px
  - Code/eyebrows/labels: JetBrains Mono 400, 12–13px

Button rules:
  Primary (marketing): bg #171717, text white, border-radius 100px, padding 0 12px, height ~48px
  Secondary (marketing): bg white, text #171717, border: 1px solid #ebebeb, same pill shape
  Nav Sign Up: bg #171717, text white, border-radius 6px, height 28px
  Nav Log In: bg white, text #171717, border-radius 6px, height 28px

Page band rhythm: canvas-soft (#fafafa) → canvas (#fff) → primary dark (#171717) polarity-flip

── TECH STACK ────────────────────────────────────────────────

Backend: Node.js + Express
Database: PostgreSQL (or SQLite for MVP) via Prisma ORM
Frontend: Vanilla HTML/CSS/JS or React (your choice)
AI: OpenAI API (GPT-4o) for failure analysis
Auth: Simple email/password or magic link

── AI INTEGRATION ────────────────────────────────────────────

When a new postmortem is submitted, send the story + metrics to OpenAI GPT-4o:

System prompt: "You are an expert startup analyst. Given a startup's postmortem, extract: 1) Primary root cause (one sentence), 2) Contributing factors (3-5 bullet points), 3) Category tags from this list: [PMF mismatch, Cash flow, Team conflict, Market timing, Competition, Tech debt, Regulatory, Execution, Customer acquisition, Pivot failure]. Respond in JSON."

Store the AI analysis alongside the postmortem in the database.
Display it in the dark code-editor-mockup card on the postmortem page.
